## Disclaimer

⚠️ All our plugins must be used at your own risk.

## Installation

Add `https://xiv.starry.blue/plugins/master.json` to your custom plugin repositories.

After the Dalamud API level is updated, plugins in the Stable channel will not be updated immediately. If you need updates promptly, configure your Dalamud to receive updates from the Testing channel.
